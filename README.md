# Opis projektu c++
Aleksadra Polek 204150

Projekt to gra zręcznościowa, stworzona w języku C++ z wykorzystaniem biblioteki graficznej SFML. Celem gracza jest zbijanie kolorowych bloków za pomocą piłki odbijanej przez ruchomą paletkę.
Gra zawiera punktację, losowość kierunku piłki na starcie, system combo, różne kształty piłki oraz możliwość zapisu i odczytu stanu gry.

Funkcjonalność:
Menu Główne: Nawigacja klawiaturą, opcje: Nowa Gra, Zmiana Kształtu, Wczytaj Grę, Wyniki, Wyjście.

Zróżnicowane Bloki: 8 rzędów bloków o różnej wytrzymałości (HP), bloki zmieniają kolor po każdym trafieniu.:

*Fioletowe: 4 uderzenia.

*Niebieskie: 3 uderzenia.

*Żółte: 2 uderzenia.

*Czerwone: 1 uderzenie.

System Wyboru Kształtu: Gracz może wybrać w menu kształt piłki: kulę, gwiazdę i serce.

System Punktacji i Combo:

Podstawowe punkty za zbijanie bloków.

Dynamiczny licznik Combo: Seria trafień bez dotykania paletki zwiększa mnożnik punktów. Napis z informacją pojawia się na górze ekranu.

System High Score:

Zapis najlepszego wyniku do pliku.

Wyświetlanie rekordu oraz ostatniego wyniku w zakładce wyniki.

Zapis i Odczyt Stanu Gry:

Zapis do pliku (F5) - pozwala kontynuować grę po ponownym uruchomieniu programu.

Zapis do pamięci RAM (P) - szybki zapis w trakcie sesji.

Wczytanie gry (L lub z Menu).

Oprawa Wizualna: Tło gradientowe, dynamiczne teksty.
